JQuery Quiz - 1

The jQuery function is the value of jQuery or of $. This is the function that creates jQuery objects and registers handlers to be invoked when the DOM is ready; it also serves as the jQuery namespace.

A jQuery object is an object returned by the jQuery function. A jQuery object represents a set of document elements and can also be called a "jQuery result", a "jQuery set", or a "wrapped set"
